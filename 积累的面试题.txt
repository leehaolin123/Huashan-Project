当你写 int x = 5; 时，C# 编译器 (csc) 知道 int 对应于 CTS 中的 System.Int32 类型

当被问 .NET vs C# 类型 时，强调 “语言别名与框架底层类型的关系”
当被问 .NET vs C++ 类型 时，强调 “托管与非托管、内存管理、类型安全” 的差异。
